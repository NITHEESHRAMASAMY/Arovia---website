## AROVIA WEBSITE

A modern, responsive e-commerce website for Arovia products built with React, Vite, and Tailwind CSS.

### 🎯 Features

- **Responsive Design**: Fully responsive UI that works on desktop, tablet, and mobile devices
- **Product Showcase**: Beautiful product cards and modal dialogs for detailed product information
- **Contact Section**: Easy-to-use contact form for customer inquiries
- **Instagram Integration**: Showcase Instagram content and drive social media engagement
- **Smooth Scrolling**: Scroll-to-top functionality for enhanced user experience
- **Toast Notifications**: Real-time feedback for user actions
- **Modern UI Components**: Custom badge, button, dialog, and toast components

### 🛠️ Tech Stack

- **Frontend Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Component Library**: Custom UI components with Radix UI patterns
- **Package Manager**: npm/yarn

### 🚀 Getting Started

#### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

#### Installation

1. Clone the repository:
```bash
git clone https://github.com/NITHEESHRAMASAMY/Arovia---website.git
cd Arovia_2
```

2. Install dependencies:
```bash
npm install
cd apps/web
npm install
```

3. Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### 📦 Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
